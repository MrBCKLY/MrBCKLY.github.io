Pseudocode is a method of writing up a set of instructions for a computer program using plain English. It is commonly used to plan a program before any coding starts or to debug a logical error.

Pseudocode is language independent, which means that anyone that knows how to program should be able to read pseudocode and have an idea of what the writer was trying to achieve.

You should be able to read these algorithms and calculate what will have been outputted once the program is run. The outputs concatenate together, which means they are joined end to start with no additional characters inserted between them, such as a space.

                            OUTPUT 'Hello'
OUTPUT 'world!'
This would result as 'HelloWorld!'
